RNAワールドを作れ！

生命起源ゲーム ルール説明書

ゲームの目的

プレイヤーは生命の材料を集め、「脂質二重膜」「タンパク質」「RNA」の3つの生命構成要素を、誰よりも早く完成させることを目指します。最初にすべてを完成させたプレイヤーが勝者です！

完成物と効果

1. 脂質二重膜（細胞膜）

生命の器、すべてはここから始まる。

必要材料： 脂肪酸 x 4枚

完成効果： バリア形成！ これ以降、相手の妨害カード（UV、シアン）の効果を受けなくなります。

2. タンパク質（酵素）

生命活動を助ける働き者。

必要材料： アミノ酸 x 3枚

完成効果： 反応促進！ RNAを合成する「脱水縮合」を、1ターンに1回多く行えるようになります。（合計2回）

3. RNA（遺伝子と酵素）

自己増殖する生命の中心物質。

必要材料： リボース x 2枚 + 核酸塩基 x 2枚 + リン酸 x 1枚

完成効果： 自己増殖！ 山札から好きな材料カードを好きなだけ手札に加えられます。これにより、他の完成も一気に進むでしょう。

材料カード (全5種)

→ RNA

環境カード (全4種)

【隕石】

山札からアミノ酸・脂肪酸・リン酸を引く or 材料カード設置上限を3枚にする

山札から「アミノ酸」「脂肪酸」「リン酸」を1枚ずつ手札に加える。またはこのターンに設置できる材料カードを3枚にする。

【UV（紫外線）】

阻害+相手の手札1枚を山札に戻す

相手の場にある材料カード1枚を一時使用不能にする。ただし、完成済みまたは修復済みのカードには使えない。その後、相手の手札の材料カードから一枚を引き、相手の山札に戻す。

【ホウ酸】

脱水縮合+山札からリボースを1枚引く

自分の場の材料カードに挟まれた水カードを裏返し、結合状態に変える。その後、山札から「リボース」1枚を手札に加える。

【シアン化水素】

阻害+核酸塩基を1枚引く

相手の場にある材料カード1枚を一時使用不能にする。ただし、完成済みまたは修復済みのカードには使えない。その後、山札から「核酸塩基」1枚を手札に加える。

ゲームの進め方

準備：材料カード・環境カードをそれぞれよく混ぜて別の山札とし、各プレイヤーは材料カード2枚、環境カード3枚を引き、手札を5枚とします。

自分のターンが来たら、以下の1〜3を順番に行います。

  1. ドローフェイズ：材料カード、環境カードを1枚ずつ引いて手札に加えます。
  2. アクションフェイズ：手札からカードを好きなだけプレイできます。
    • 環境カードをつかう：手札の「環境カード」を使い、効果を発動する。1ターンにつき1枚まで使用でき、使ったカードは捨て札にする。
    • 材料カードを場に出す：手札の「材料カード」を1ターンにつき2枚まで自分の場に出すことができる。
    • 脱水縮合を行う：自分の場の材料カードに挟まれた水カードを裏返し、2枚の材料カードを結合状態に変える。これを「脱水縮合」と呼び、1ターンに1回までできる。（タンパク質完成後は2回）
    • 再合成する：自分の場の材料カードのうち、シアンあるいはUVにより阻害されているカードを1枚修復できる。修復のためには、同じ材料カードをシアンあるいはUVカードの上に重ねる必要がある。
    • カードを交換する：自分の手札の材料カード1枚を捨て、材料カードの山札から1枚を引く。
  3. エンドフェイズ：ターンを終了し、次のプレイヤーに交代します。
